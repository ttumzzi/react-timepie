# Time Pie 🧀

쉽고 간결하게 시간표를 짜도록 도와줍니다. 드래그를 이용해 시간을 선택할 수 있습니다.

### 📌 기능
- [x] 10분 단위로 계획표 추가 가능
- [ ] 드래그로 영역 지정 후 계획 작성
- [ ] 계획표 선택 시 우측에 상세페이지 오픈
- [ ] 현재 시간 부분에 하이라이팅
- [ ] 계획표 여러 개 추가 가능
- [ ] 계획표를 이미지 파일로 변환 가능

### 📌 사용 기술
- React.js
- HTML canvas
- recoil.js
